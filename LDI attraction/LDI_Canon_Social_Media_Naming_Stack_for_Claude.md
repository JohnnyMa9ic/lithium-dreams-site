# 11. YouTube / Video Architecture

Recommendation:

Start with **one main YouTube channel** under the LDI umbrella, then organize content into named shows/playlists.

Avoid creating too many separate channels at the beginning.

## Main Channel Recommendation

```text
Lithium Dreams Broadcasts
```

Possible handles:

```text
@LithiumDreamsBroadcasts
@LDIBroadcasts
@LithiumDreamsTV
```

## Shows / Playlists

```text
The Midnight Visitor Log
Operator Notes
Object Lab Dispatch
```

## Why this works

One main channel prevents fragmentation while still allowing clear content separation.

- The Midnight Visitor Log attracts curiosity and story-driven viewers.
- Operator Notes attracts builders, operators, and systems people.
- Object Lab Dispatch connects product creation, behind-the-scenes work, and store launches.

If the audiences eventually become large and distinct enough, the channels can be separated later.

# 13. Brand Boundary Rules

## Do

- Keep The Midnight Visitor Log atmospheric.
- Keep Operator Notes practical.
- Keep The Last Roadside Gift Shop product-rich.
- Keep LDI Object Lab tactile and experimental.
- Keep Lithium Dreams Industries as the parent brand.
- Let the weirdness create memory and differentiation.
- Let the corporate side create trust and conversion.

## Do Not

- Make the Attraction corporate.
- Make the corporate side goofy.
- Turn The Midnight Visitor Log into generic horror.
- Turn Operator Notes into generic AI influencer content.
- Mix all products into one confusing undifferentiated store.
- Overexplain the lore before people are emotionally interested.
- Create too many social channels before content volume justifies it.
