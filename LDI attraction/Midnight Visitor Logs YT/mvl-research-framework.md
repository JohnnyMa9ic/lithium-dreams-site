# Midnight Visitor Logs — Research & Script Production Framework
### Lithium Dreams TV | Repeatable Workflow v1.0

---

## Purpose

This framework documents the repeatable process for researching, structuring, and scripting a *Midnight Visitor Logs* episode on any paranormal, fringe science, conspiracy, or cryptid subject. It is designed to be executed by an AI agent (e.g., via Hermes / Claude Code) or used manually as a production checklist. Every deliverable produced follows the same architecture so episodes feel consistent while content stays unique.

---

## Phase 0: Pre-Research Definition

Before any searching begins, define the episode's scope parameters. These answers drive every subsequent decision.

| Parameter | Question to Answer | Example (Mothman) |
|-----------|-------------------|-------------------|
| **Core Subject** | What is the central entity, event, or phenomenon? | Mothman — winged humanoid cryptid |
| **Anchor Geography** | Where is the primary incident location? | Point Pleasant, West Virginia |
| **Time Spine** | What is the full documented timeline? | 1966–2021 |
| **Secondary Clusters** | Are there related geographical or temporal clusters? | Chicago metropolitan area 2011–2021 |
| **Climactic Event** | Is there a tragic, dramatic, or decisive event that anchors the narrative? | Silver Bridge collapse, December 15, 1967 |
| **Theory Landscape** | What are the 2–4 dominant explanatory frameworks? | Cryptid / Mass Hysteria / Ultraterrestrial |
| **Emotional Core** | What human experience does this subject tap into? | Fear of the unknown, warning signs, disaster and grief |
| **Target Episode Tone** | Ominous? Playful? Academic? Investigative? | Investigative + ominous + wonder |

## Phase 1: Parallel Research Tracks

Run all five tracks simultaneously. Each is a discrete search batch. Save raw results to workspace files for downstream reference.

### Track 1 — The Timeline
**Goal:** Build a full chronological event log with dates, names, and locations.

Search targets:
- `[Subject] complete timeline sightings [start year] [end year]`
- `[Subject] first sighting origin incident [location]`
- `[Subject] last known sighting`

Deliverable: A structured table with columns: `Date | Witness Name(s) | Location | Description | Source`

Minimum quality bar: 10+ dated events with named or credentialed witnesses.

---

### Track 2 — Witness Accounts
**Goal:** Collect 10–20 individual eyewitness reports with specificity.

Search targets:
- `[Subject] witness accounts specific descriptions [location]`
- `[Subject] [location] individual sightings [year]`
- `[Subject] MUFON reports / Lon Strickler / NUFORC` (adapt to relevant registry)

Quality criteria for a usable witness account:
- [ ] Has a date (or approximate date)
- [ ] Has a named witness OR a credentialed anonymous witness (law enforcement, aviation, medical, trade)
- [ ] Has a specific physical description (size, color, eye detail, sound, behavior)
- [ ] Has a direct quote, if available
- [ ] Has a source URL that can be cited

If fewer than 10 accounts meet the bar, dig into primary investigators' blogs, MUFON case numbers, and paranormal registry databases.

---

### Track 3 — The Anchor Event
**Goal:** If a climactic event exists (bridge collapse, disaster, disappearance, death), obtain the factual, official record in full detail.

Search targets:
- `[Anchor event] official investigation findings [agency]`
- `[Anchor event] engineering / forensic cause [year]`
- `[Anchor event] [NTSB / NIST / coroner / FBI] report`
- `[Anchor event] death toll timeline survivor accounts`

The factual record must be separated from paranormal interpretation. Document:
1. What officially happened (cause, mechanism, timeline)
2. What the paranormal narrative claims
3. What evidence connects or fails to connect the two

---

### Track 4 — Geographical Clusters
**Goal:** Map where incidents concentrated and identify any spatial patterns.

Search targets:
- `[Subject] sightings map geographical distribution`
- `[Subject] [location] cluster concentration [year]`
- `[Subject] sightings spread states / countries`

Deliverable: A cluster table with columns: `Location Name | Type | # of Documented Sightings | Notable Events | Notes`

Look for: proximity to water, industrial sites, transportation hubs, military installations, or cultural/historical sites of significance.

---

### Track 5 — Theory Landscape
**Goal:** Document 3 distinct explanatory theories with supporting evidence and counter-evidence for each.

Standard theory archetypes (adapt to subject):

| Archetype | Description |
|-----------|-------------|
| **Biological / Physical** | A real, material explanation (misidentified animal, natural phenomenon, human actor) |
| **Psychosocial** | A psychological or sociological explanation (mass hysteria, media amplification, confirmation bias, cultural projection) |
| **Paranormal / Metaphysical** | A supernatural, ultraterrestrial, interdimensional, or extraterrestrial explanation |

For each theory, research and document:
- Core argument in 1–2 sentences
- 3–5 supporting arguments with cited sources
- 3–5 counter-arguments with cited sources
- Key advocates (researchers, institutions, investigators)

## Phase 2: Source Quality Scoring

Before writing begins, score each source used. Prioritize high-quality sources for hard factual claims; lower-quality sources are acceptable for witness narrative and theory framing.

| Tier | Source Type | Examples | Use For |
|------|-------------|----------|---------|
| **Tier 1** | Official government / scientific body | NTSB reports, USGS, peer-reviewed journals | Factual anchor events, engineering causes |
| **Tier 2** | Established news / academic | New York Times, university papers, National Audubon Society | Historical record, expert opinion |
| **Tier 3** | Dedicated paranormal researchers | Lon Strickler (Phantoms & Monsters), Singular Fortean Society, MUFON | Witness aggregation, case indexing |
| **Tier 4** | Enthusiast / blog | BeyondHaunted, paranormal wikis, YouTube summaries | Cross-reference, color, narrative detail |

**Rule:** Never use a Tier 4 source as the sole citation for a factual claim. Always corroborate with Tier 1–2.

## Phase 3: Narrative Architecture

Every *Midnight Visitor Logs* episode follows a **three-act ominous travel log** structure. Map research to acts before writing begins.

### Act I — The Place and the Before
- Establish the setting as a character: geography, history, atmosphere, what it felt like before the phenomenon arrived
- Introduce the anchor location in its mundane reality — then break it
- End Act I on the **inciting incident**: the first major documented sighting or event

**Emotional target:** Familiarity becoming unease. The audience should feel like they know the place right before they realize they don't.

---

### Act II — The Wave
- Run through the sighting timeline and key witness accounts
- Introduce the phenomenon's behavior patterns (what it does, how it moves, who sees it)
- Escalate toward the climactic event if one exists
- Introduce competing theories mid-act — not as resolution, but as texture and tension
- End Act II on the climactic event (the collapse, the disappearance, the final sighting)

**Emotional target:** Building dread, interrupted by wonder. The audience should not know whether to be afraid or fascinated — ideally both.

---

### Act III — The Aftermath and the Open Door
- Document the modern echo: where does the phenomenon appear today?
- Evaluate theories without forcing a verdict
- Leave the audience in productive uncertainty — facts laid out, conclusion withheld
- Close with a line that honors the mystery without dismissing the witnesses

**Emotional target:** Reflection and hunger. The audience should leave thinking — and want to come back.

## Phase 4: Tagline and Catchphrase Generation

For each episode, generate:

1. **Episode-specific hook** — Opens the cold open or appears in the thumbnail. Tied to the specific subject.
2. **Series-recurring taglines** — 3–5 phrases used consistently across Midnight Visitor Logs to build brand identity.
3. **Mid-script callback lines** — Internal phrases that can be repeated across the episode for rhythm and audience identification.

**Tagline criteria:**
- Conversational, not academic
- Written in the host's voice (45-year-old Midwest man; passionate, grounded, wondering)
- Contains a tension between the factual and the mysterious
- No more than 12 words
- Memorable enough to use as a thumbnail text overlay or chapter marker

See the companion tagline document for Mothman-specific examples and series-level phrases.

## Phase 5: Script Drafting Standards

### Voice Profile
The *Midnight Visitor Logs* host voice is:
- **Age/Region:** 45, Midwest — flat vowels, plain speech, no coastal affectation
- **Register:** Conversational but deliberate; never academic, never pandering
- **Relationship to mystery:** Believer-adjacent; presents facts clearly, never demands the audience adopt a conclusion
- **Relationship to tragedy:** Respectful; human loss is never exploited for drama
- **Signature trait:** The host *wants* to believe — and that wanting is visible

### Script Format

```
[COLD OPEN]
[Short, punchy scene-setting. 30–60 seconds on camera or voiceover.]
[End on a question or image that hooks the viewer before the title card.]

[TITLE CARD / INTRO MUSIC]

[ACT I — THE PLACE]
[Narration / host on camera. ~3–5 minutes.]

[ACT II — THE WAVE]
[Narration with archive stills / witness account readings. ~8–12 minutes.]

[THEORY BREAK]
[Host directly addresses camera. Lays out the three theories in plain language. ~3–5 minutes.]

[ACT III — THE OPEN DOOR]
[Modern echo + conclusion. ~2–4 minutes.]

[OUTRO / CALLBACK LINE]
[Series signoff. Consistent every episode.]
```

### Script Technical Specs

| Element | Standard |
|---------|----------|
| Total script length | 2,500–4,000 words (15–25 min episode) |
| Sentence length | Favor short-to-medium sentences for spoken delivery |
| Paragraph length in script | 3–5 sentences max before a break or stage direction |
| Stage directions | Use `[BEAT]`, `[PAUSE]`, `[TO CAMERA]`, `[VOICEOVER]` |
| Witness quote formatting | Block-indent; introduce speaker before quote |
| Theory labels | Present in neutral language — "some say," "the skeptics argue," "others believe" |

## Phase 6: Post-Script QC Checklist

Before marking an episode complete, verify:

- [ ] Every factual claim (death tolls, dates, engineering causes, official findings) cites a Tier 1–2 source
- [ ] No single witness account is treated as definitive proof of anything
- [ ] Each of the 3 theories receives fair presentation — no theory is mocked
- [ ] The anchor event (if applicable) is handled with appropriate gravity
- [ ] The host voice is consistent throughout — no academic drift, no tabloid sensationalism
- [ ] The episode ends on productive uncertainty, not a verdict
- [ ] The series signoff line appears at the close
- [ ] Running time estimate falls within target range (read aloud at natural pace to verify)
- [ ] Taglines / callback lines are embedded at logical breakpoints

## Appendix B: Agent Prompt Templates

Use these as base prompts when delegating research phases to an AI agent.

### Track 1 — Timeline Prompt
```
Research the complete documented timeline of [SUBJECT] from [START YEAR] to [END YEAR]. 
Extract all events with: exact date (or closest approximation), 
witness names or credentialed descriptions, location (city, state, specific site), 
creature/phenomenon description, and source URL. 
Organize as a markdown table. Minimum 10 dated events. 
Save to workspace as [subject]-timeline.md.
```

### Track 2 — Witness Accounts Prompt
```
Collect 10–20 individual eyewitness accounts of [SUBJECT] from [LOCATION(S)]. 
For each account include: date, witness name or professional background, 
location, full physical description of what was seen, any direct quotes, 
and source URL. Prefer named witnesses or anonymous witnesses with verifiable 
professional credentials. Organize as a markdown table.
```

### Track 5 — Theory Prompt
```
Document three distinct explanatory theories for [SUBJECT]: 
(1) a biological or physical explanation, 
(2) a psychosocial or mass hysteria explanation, 
(3) a paranormal or extraterrestrial explanation. 
For each theory: state the core argument, list 3–5 supporting arguments 
with cited sources, list 3–5 counter-arguments with cited sources, 
and name key advocates. Do not favor any theory. Present all in neutral language.
```
